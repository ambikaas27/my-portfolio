# Ambika Sharma — Developer Portfolio

🌐 **Live Demo:** [ambikaas27.github.io/my-portfolio](https://ambikaas27.github.io/my-portfolio)

A personal portfolio website built with React showcasing my projects, skills and experience as a Full Stack Developer.

## ✨ Features

- 🌙 Dark / Light mode toggle
- ⌨️ Typing animation in hero section
- 🎯 Scroll animations (AOS)
- 📊 Skill progress bars
- 📬 Working contact form (EmailJS)
- 📄 Resume download button
- 🖱️ Custom cursor
- ⬆️ Back to top button
- 🔆 Active navbar highlighting
- 💫 Loading screen
- 📱 Fully responsive design

## 🛠 Tech Stack

- **Frontend:** React.js, CSS
- **Animations:** AOS (Animate On Scroll)
- **Email:** EmailJS
- **Deployment:** GitHub Pages

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/ambikaas27/my-portfolio.git

# Go into the project
cd my-portfolio

# Install dependencies
npm install

# Start development server
npm start
```
